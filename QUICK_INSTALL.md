# 🚀 Focus Lens - Quick Install (30 seconds)

## Download & Extract
1. Download `Focus-Lens-v1.0.zip`
2. Double-click to extract
3. Open the `Focus-Lens` folder

## Install (Choose ONE method)

### 🟢 EASIEST: Right-Click Method
```
Right-click "Focus Lens.app" → Select "Open" → Click "Open"
```

### 🔵 ALTERNATIVE: Terminal Script
```bash
cd /path/to/Focus-Lens  # (drag folder here to auto-fill)
bash install.sh
```

## Grant Permissions
When prompted, allow:
- ✅ Camera (for eye tracking)
- ✅ Screen Recording (for overlay)

## ⚠️ Security Warning?
**This is normal!** The app is safe but unsigned (standard for beta apps).

The right-click method bypasses this warning in 1 click.

---

## 🎯 Using the App

- **Enable/Disable:** Click power button
- **Eye Tracking:** Click "Eye" button → "Calibrate"
- **Adjust Effects:** Use sliders
- **Move Control Bar:** Drag it anywhere

---

## 💡 Troubleshooting

| Problem | Solution |
|---------|----------|
| "App is damaged" | Run: `xattr -cr "Focus Lens.app"` |
| Eye tracking not working | Enable Camera in System Settings → Privacy |
| Overlay invisible | Enable Screen Recording in System Settings |
